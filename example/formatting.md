# How To Use GitHub for Bosses

For doing writeups, there's the same basic markdown syntax present in Discord to be utilized

```
# Big header
## Medium header
### Small header
**Bold text**
*Italic text*
~~Strikethrough text~~
[Link text](Thing being linked to)
```
